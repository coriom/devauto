# Format du fichier questions.txt (QCM)
# - Les lignes commençant par '#' sont des commentaires et peuvent être ignorées.
# - Chaque question est séparée par une ligne contenant exactement: ---
# - Structure d'un bloc question:
#     Q: <intitulé de la question>
#     A) <option A>
#     B) <option B>
#     C) <option C>
#     D) <option D>
#     ANS: <A|B|C|D>
#
# Exemple:
# Q: ...
# A) ...
# B) ...
# C) ...
# D) ...
# ANS: B
---
Q: Quel langage est principalement utilisé pour écrire des scripts d'automatisation et des analyses de données simples ?
A) Python
B) HTML
C) SQL
D) CSS
ANS: A
---
Q: Dans un réseau, à quoi sert principalement un routeur ?
A) Stocker des fichiers sur un disque partagé
B) Diriger des paquets entre différents réseaux
C) Augmenter la résolution d'un écran
D) Refroidir un processeur
ANS: B
---
Q: Quelle structure de données fonctionne selon le principe FIFO (First In, First Out) ?
A) Une pile (stack)
B) Un ensemble (set)
C) Une file (queue)
D) Un arbre (tree)
ANS: C
